# Add Quote

Add Quote SwiftUI App with MVVM
 
 ### Features
  	• SwiftUI
 	• Project Organization & Architecture
	• MVVM 
	• Support Dark Mode & light mode 
 	• Save and persist data with UserDefaults
	• Downloading & Caching Image
	• Custom Animations
